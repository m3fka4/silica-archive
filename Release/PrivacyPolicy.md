# Silica Privacy Policy Draft

Silica is designed as a local-first macOS utility.

- Silica does not upload files for compression, extraction or image optimization.
- Silica does not require cloud processing.
- Archive passwords, if saved by the user, must be stored in Keychain.
- Private Mode avoids operation history and removes image metadata where possible.
- Finder, Quick Panel and Menu Bar actions process files locally on the user's Mac.

This draft must be reviewed before public distribution.
