Silica v0.1.0-alpha

First public alpha of Silica, a native macOS compressor for archives, images and everyday files.

What works:
- Native SwiftUI macOS app
- Drag & drop
- ZIP create/list/test/extract
- Image optimization
- Silica Lens analysis
- Menu bar actions
- Floating Quick Panel
- Profiles and history

Known limitations:
- Finder extension is still on the roadmap
- RAR/7Z support depends on local backend availability
- This is an early alpha build
